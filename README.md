# Pokemon Trainers

The registration form for new Pokemon Trainers.

## Used Stuff

- Next.js
- React.js
- TypeScript
- Jest
- Atomic Design
- Material UI

## Node version 18.18.2

## Run application

1. Install dependencies
   `yarn`
2. Run test environment
   `yarn dev`

## Run unit tests

`yarn test`

OR with observables

`yarn test:watch`
